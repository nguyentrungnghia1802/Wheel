# Wheel of Names - Clone

Đây là bản clone của trang web wheelofnames.com với đầy đủ chức năng quay vòng ngẫu nhiên.

## Tính năng

✅ **Vòng quay động**
- Vòng quay với nhiều màu sắc đẹp mắt
- Chuyển động quay đều, chậm rãi khi idle
- Quay nhanh và dần chậm lại khi spin
- Shadow và hiệu ứng 3D

✅ **Mũi tên thông minh**
- Tự động đổi màu theo phần đang trỏ
- Animation mượt mà

✅ **Editor thông minh**
- Thêm/xóa/chỉnh sửa entries trực tiếp
- Tự động cập nhật vòng quay real-time
- Hỗ trợ Shuffle và Sort

✅ **Kết quả**
- Popup hiển thị người thắng cuộc
- Hiệu ứng confetti rơi
- Nút Remove: xóa người thắng khỏi danh sách
- Nút Close: giữ nguyên danh sách

✅ **Responsive**
- Tự động điều chỉnh theo màn hình
- Hỗ trợ mobile và desktop

## Cách sử dụng

1. Mở file `index.html` trong trình duyệt
2. Chỉnh sửa danh sách tên ở bảng bên phải
3. Click vào vòng quay hoặc nút "Click to spin" để quay
4. Chờ vòng quay dừng lại và xem kết quả
5. Chọn "Remove" để xóa người thắng hoặc "Close" để giữ nguyên

## Files

- `index.html` - File HTML chính
- `styles.css` - File CSS với đầy đủ styling
- `wheel.js` - File JavaScript xử lý logic vòng quay
- `qcao.jpeg` - Ảnh quảng cáo placeholder

## Customization

Bạn có thể tùy chỉnh:
- Màu sắc vòng quay trong `wheel.js` (dòng 9)
- Tốc độ quay trong `wheel.js` (dòng 12-14)
- Font chữ trong `styles.css`
- Layout và colors trong `styles.css`
